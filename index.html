Welcome to TSC

TSC Means TopSocialConnect 🚀

TopSocialConnect is my personal social media platform where users can connect, share, interact, and communicate in real time.

Thank you Mrs Monique!

This app is built using JavaScript, HTML, and CSS, and it simulates a modern social network experience inspired by Instagram-style interaction.

✨ What you can do

  • 👤 Create your own account (Sign Up / Login)
  • 🖼 Upload a profile picture
  • 📝 Create posts with text and images
  • ❤️ Like posts
  • 💬 Comment on posts
  • 👥 Follow / Unfollow other users
  • 🔔 Receive notifications
  • 📸 View stories
  • 👤 Visit user profiles
  • 💌 Chat with other users
  • 🔍 Search users and posts
  • 🏠 Navigate through Home / Search / Profile

🌟 App Goal

SocialConnect is designed to simulate a real modern social network inspired by Instagram-style interaction, helping users connect and share.

Start by creating an account and posting your first update!
